# Writing Tools

Tools and resources for improving technical writing, grammar, and content creation.

## Writing & Grammar Tools

### Browser Extensions
- [Grammarly](https://www.grammarly.com/) - AI-powered writing assistant
- [LanguageTool](https://languagetool.org/) - Grammar and style checker
- [QuillBot](https://quillbot.com/) - Paraphrasing and writing tool

### Editors
- [HackMD](https://hackmd.io/) - Collaborative markdown editor
- [Lapce](https://github.com/lapce/lapce) - Fast code editor
- [Mark Text](https://github.com/marktext/marktext) - Real-time markdown editor
- [Swimm](https://swimm.io/) - Documentation platform for code
- [Visual Studio Code](https://code.visualstudio.com/) - Popular code editor
- [Zed](https://zed.dev/) - High-performance code editor
- [Zettlr](https://www.zettlr.com/) - Academic writing and note-taking

### Knowledge Base
- [Logseq](https://logseq.com/) - Privacy-first, open-source knowledge base
- [Obsidian](https://obsidian.md) - Knowledge management and note-taking

## Content Creation

### Mockup & Visualization
- [Animockup](https://github.com/alyssaxuu/animockup) - Create animated mockups
- [Excalidraw](https://excalidraw.com/) - Hand-drawn like diagrams
- [Markmap](https://markmap.js.org/) - Visualize markdown as mind maps

### Screenshots & Screencasts
- [Carbon](https://carbon.now.sh) - Beautiful code screenshots
- [Flameshot](https://github.com/lupoDharkael/flameshot) - Screenshot tool
- [shot-scraper](https://github.com/simonw/shot-scraper) - Command-line screenshots
- [Shottr](https://shottr.cc/) - macOS screenshot annotation

### Screencast Tools
- [Asciinema](https://asciinema.org/) - Record terminal sessions
- [Codio](https://present.readthedocs.io/en/latest/codio.html) - Interactive coding presentations
- [Editly](https://github.com/mifi/editly) - Programmatic video editing
- [Kap](https://github.com/wulkano/kap) - Screen recorder for macOS
- [Peek](https://github.com/phw/peek) - GIF screen recorder for Linux
- [Screenity](https://github.com/alyssaxuu/screenity) - Browser screen recorder
- [Terminalizer](https://github.com/faressoft/terminalizer) - Record terminal sessions as GIF
